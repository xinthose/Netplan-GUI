Bridge

Treat both ethernet ports of the computer as a switch. The MAC Address of the first ethernet port on the computer. Error: MAC Address LAN 1 is required. The MAC Address of the second ethernet port on the computer. Error: MAC Address LAN 2 is required. The IP Address of the network's router. Only one Gateway is allowed. The IP Address of the network's DNS server (default: 8.8.8.8). The IP Address of the network's backup DNS server (default: 8.8.4.4). IP addresses of the bridge (can add multiple). Error: Address is required. {{ getSubnetMask(dataItem.cidr)?.subnetMask }}

Ethernet 1

Port LAN1 on the computer. The MAC Address of the first ethernet port on the computer. Error: MAC Address is required. The IP Address of the network's router. Only one Gateway is allowed. The IP Address of the network's DNS server (default: 8.8.8.8). The IP Address of the network's backup DNS server (default: 8.8.4.4). IP addresses of the port. Error: Address is required. {{ getSubnetMask(dataItem.cidr)?.subnetMask }}

Ethernet 2

Port LAN2 on the computer. The MAC Address of the second ethernet port on the computer. Error: MAC Address is required. The IP Address of the network's router. Only one Gateway is allowed. The IP Address of the network's DNS server (default: 8.8.8.8). The IP Address of the network's backup DNS server (default: 8.8.4.4). IP addresses of the port. Error: Address is required. {{ getSubnetMask(dataItem.cidr)?.subnetMask }}

Wi-Fi

Wi-Fi network that the computer connects to (wlp1s0). The IP Address of the network's router. Only one Gateway is allowed. The IP Address of the network's DNS server (default: 8.8.8.8). The IP Address of the network's backup DNS server (default: 8.8.4.4). Name of the Wi-Fi network to connect to (Service Set Identifier). Error: SSID is required. Passphrase for WPA2 authentication. If not given, the network is assumed to be open. Other authentication modes are not currently supported. IP addresses of the network. Error: Address is required. {{ getSubnetMask(dataItem.cidr)?.subnetMask }}
